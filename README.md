# stock-performance-dashboard
Developed an interactive stock market dashboard using Power BI and Python to analyze price trends, returns, and volatility. Applied normalization techniques to compare asset performance and generated insights on risk and return.

Objectives
1.Compare stock performance over time
2.Analyze risk using volatility
3.Evaluate returns using percentage change
4.Build an interactive dashboard for decision-making

Tools Used
1.Python (pandas, numpy, yfinance)
2.Power BI

Dashboard Features
1.Latest price KPI cards
2.Average returns and volatility metrics
3.Normalized performance comparison
4.Monthly returns analysis
5.Trading volume visualization
6.Interactive date filtering


Key Insights
Tesla shows higher average returns compared to Apple
Tesla also exhibits higher volatility, indicating greater risk
Apple demonstrates more stable and consistent performance
